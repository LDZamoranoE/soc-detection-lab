# SOC Detection Lab

This project documents a personal cybersecurity lab used to simulate attacks and analyze network traffic like a SOC analyst.

## Tools used

- Kali Linux
- Wireshark
- Nmap
- Burp Suite

## Objectives

- Generate suspicious traffic
- Capture packets
- Analyze network behavior
- Document potential detections

## Simulated attacks

- Network scanning
- HTTP enumeration
- Suspicious traffic generation
